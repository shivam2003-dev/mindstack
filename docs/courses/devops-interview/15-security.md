Security is critical in DevOps. This chapter covers security best practices and DevSecOps.

!!! tip "Interview Focus"
    Understand security principles, secrets management, and security automation.

## DevSecOps Principles

- Security as code
- Shift left security
- Automated security testing
- Continuous security monitoring

## Best Practices

- Least privilege
- Secrets management
- Image scanning
- Network policies
- Regular updates
- Security audits

## Secrets Management

- Use secret managers (AWS Secrets Manager, HashiCorp Vault)
- Never commit secrets
- Rotate secrets regularly
- Use environment variables carefully
