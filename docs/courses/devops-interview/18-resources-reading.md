## üìö Recommended Books

### DevOps Fundamentals

1. **"The Phoenix Project" by Gene Kim, Kevin Behr, George Spafford**
   - DevOps culture and principles
   - Real-world scenarios
   - Essential for understanding DevOps mindset

2. **"The DevOps Handbook" by Gene Kim, Jez Humble, Patrick Debois, John Willis**
   - Comprehensive DevOps guide
   - Best practices and patterns
   - Implementation strategies

3. **"Accelerate" by Nicole Forsgren, Jez Humble, Gene Kim**
   - Research-based DevOps practices
   - Metrics and measurements
   - Performance improvement

### Linux & System Administration

4. **"Linux Command Line and Shell Scripting Bible" by Richard Blum, Christine Bresnahan**
   - Complete Linux command reference
   - Shell scripting mastery
   - Practical examples

5. **"UNIX and Linux System Administration Handbook" by Evi Nemeth, Garth Snyder, Trent R. Hein**
   - Comprehensive system administration
   - Production-ready knowledge
   - Troubleshooting guide

### Container Technologies

6. **"Docker Deep Dive" by Nigel Poulton**
   - Complete Docker guide
   - Advanced concepts
   - Production practices

7. **"Kubernetes: Up and Running" by Kelsey Hightower, Brendan Burns, Joe Beda**
   - Kubernetes fundamentals
   - Practical examples
   - Production deployment

8. **"The Kubernetes Book" by Nigel Poulton**
   - Comprehensive K8s guide
   - Clear explanations
   - Real-world scenarios

### CI/CD & Automation

9. **"Continuous Delivery" by Jez Humble, David Farley**
   - CI/CD principles
   - Deployment strategies
   - Automation best practices

10. **"Infrastructure as Code" by Kief Morris**
    - IaC patterns and practices
    - Terraform, Ansible, CloudFormation
    - Infrastructure management

### Cloud & AWS

11. **"AWS Certified Solutions Architect Study Guide" by Ben Piper, David Clinton**
    - AWS services deep dive
    - Architecture patterns
    - Certification prep

12. **"Cloud Native Patterns" by Cornelia Davis**
    - Cloud-native design
    - Microservices patterns
    - Modern architectures
