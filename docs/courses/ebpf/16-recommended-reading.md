## Books

???+ "eBPF Books"
    1. [BPF Performance Tools](https://www.brendangregg.com/bpf-performance-tools-book.html) by Brendan Gregg - Comprehensive eBPF guide
    2. [Linux Observability with BPF](https://www.oreilly.com/library/view/linux-observability-with/9781492050193/) by David Calavera, Lorenzo Fontana
    3. [Systems Performance](https://www.brendangregg.com/sysperfbook.html) by Brendan Gregg - Includes eBPF chapters

???+ "Linux Kernel Books"
    1. [Understanding the Linux Kernel](https://www.oreilly.com/library/view/understanding-the-linux/0596005652/) - Kernel internals
    2. [Linux Device Drivers](https://lwn.net/Kernel/LDD3/) - Kernel programming
    3. [Professional Linux Kernel Architecture](https://www.wrox.com/WileyCDA/WroxTitle/Professional-Linux-Kernel-Architecture.productCd-0470343435.html) - Kernel architecture

## Official Documentation

???+ "Kernel Documentation"
    1. [BPF Documentation](https://www.kernel.org/doc/html/latest/bpf/) - Official kernel BPF docs
    2. [BPF Design Q&A](https://www.kernel.org/doc/html/latest/bpf/bpf_design_QA.html) - Design decisions
    3. [BPF Helper Functions](https://www.kernel.org/doc/html/latest/bpf/helpers.html) - Helper reference

???+ "Tool Documentation"
    1. [BCC Documentation](https://github.com/iovisor/bcc/blob/master/docs/reference_guide.md) - BCC reference
    2. [libbpf Documentation](https://github.com/libbpf/libbpf) - libbpf guide
    3. [bpftool Manual](https://www.mankier.com/8/bpftool) - bpftool reference

## Online Guides

???+ "Tutorials"
    1. [XDP Tutorial](https://github.com/xdp-project/xdp-tutorial) - Comprehensive XDP guide
    2. [eBPF.io Learning](https://ebpf.io/learn/) - Learning resources
    3. [Cilium eBPF Guide](https://docs.cilium.io/en/stable/bpf/) - Cilium's eBPF guide

???+ "Reference Guides"
    1. [eBPF Helper Functions](https://man7.org/linux/man-pages/man7/bpf-helpers.7.html) - Helper man page
    2. [BPF System Call](https://man7.org/linux/man-pages/man2/bpf.2.html) - bpf() syscall
    3. [BPF Maps](https://www.kernel.org/doc/html/latest/bpf/maps.html) - Map types
